Excel to `home-data.json` Converter

Instructions

  1. Important: Make sure your Excel file's first row contains these exact column headers: Vendor, component/product, type, and info. Any other columns in the Excel file will be ignored.
  2. Click "Choose File" and select your Excel (.xlsx, .xls, .csv) file.
  3. The correctly formatted JSON will appear in the text box below.
  4. Click the "Copy JSON" button and paste the content into your home-data.json file, replacing the old data.
